# WebTag

## 1.プロジェクトの概要
WebTag は学習中に見つけた有益な記事を
タグを用いて管理、ピン留め出来るアプリです


## 2.プロジェクトの目的
汚くなりがちなお気に入りバーや、大量のショートカットアイコンを綺麗にし、
さらに、後から目的の記事を検索しやすくすることを目的とします


## 3.機能一覧

### 記事管理
- 記事の追加
- 記事の編集
- 記事の削除

### タグ管理
- 記事へのタグ付け
- タグの編集
- タグの一括編集
- タグの削除

### 検索機能
- キーワード検索
- 記事の一覧表示
- タグ検索
- タグの一覧表示
